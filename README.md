# Markdown Editor Pro

An Obsidian-like WYSIWYG Markdown editor that works in both web browsers and local environments.

## Features

- ✅ WYSIWYG Markdown editing
- ✅ Real-time preview
- ✅ Local storage auto-save
- ✅ Download as Markdown
- ✅ Cross-platform (Web + Desktop)

## Getting Started

### Web Version

Open the `index.html` file in your browser.

### Local Version

Use the File System Access API to manage local Markdown files.

## Tech Stack

- **Markdown Engine:** marked.js
- **WYSIWYG Editor:** TipTap (ProseMirror-based) - Coming in MVP v2
- **Local Storage:** IndexedDB - Coming in MVP v3
- **Sync:** GitHub API - Coming in MVP v4
